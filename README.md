# Projeto Ajuda.Ai

Olá! Este é o Projeto Ajuda.Ai. O objetivo dele é ser o Patreon das Doações a Instituições/ONGs no Brasil (e futuramente no mundo!) e é totalmente **sem fins lucrativos**.

Estamos começando e vamos implementá-lo devagar e sempre :)

De onde veio a ideia? *(por Rafael Lins, criador do projeto)*
=======================

Gosto muito de coisas práticas. Obviamente sou uma pessoa que evito fazer coisas que não são práticas. E convenhamos, ajudar instituições não é nada tão pratico assim. A forma mais prática que sei é uma transferência online, usando o aplicativo do Banco no celular. Ainda assim tenho que lembrar agência e conta (tenho uma memória terrivelmente fraca, especialmente para números), e ter atenção no nome, digitar umas senhas. E, claro, ter o dinheiro na conta na hora da transferência :P

E se ajudar essas instituições pudesse ser tão simples, fácil, rápido e prático quanto fazer compras na Internet? Apenas clicar no link da instituição, se eu gostar e puder clico em "Ajudar", digo com quanto gostaria de ajudar e pronto. Forma de Pagamento? Ah... estou sem grana *agora* então põe no cartão de crédito. GG.

E foi disso que surgiu o Projeto Ajuda.Ai :)

Como ele funciona?
==================

O projeto em si é bastante simples. Instituições que quiserem participar precisarão de algumas coisas:

1. Se cadastrarem em um dos serviços de pagamento suportados (atualmente apenas PagSeguro)
2. Se necessário, criar tokens e/ou chaves de API (como se você fosse uma loja virtual)
3. Se cadastrarem no site do Projeto Ajuda.Ai
4. Configurar sua Instituição/ONG no Projeto usando os tokens e/ou chaves de API do serviço de pagamento
5. Voltar ao serviço de pagamento e configurá-lo com as URL informadas pelo Projeto Ajuda.Ai

Com isso feito sua Instituição/ONG já estará apta a receber doações!

O sistema do Projeto tem como objetivos: apresentar Instituições/ONGs que precisam de ajuda e servir como plataforma de processamento de pagamentos às Instituições/ONGs.

Isso significa que quando uma pessoa escolher doar R$ 30 para uma Instituição/ONG através desse sistema iremos criar uma ordem de pagamento no valor de R$ 30 diretamente na conta da Instituição/ONG no serviço de pagamento escolhido por ela. E quando a pessoa pagar o valor vai diretamente à conta da Instituição/ONG. De nossa parte não há nenhuma taxa, cobrança, adição ou subtração de valores. 

Sem fins lucrativos?
====================

Sim. Sem fins lucrativos. Somos uma ligação direta entre as pessoas que querem ajudar e as instituições que precisam de ajuda, sem taxas nem nada. Atualmente temos um site onde hospedamos esse sistema (depois dê uma olhada! https://ajuda.ai/) e todos os custos mensais do mesmo são bancados por mim, Rafael Lins (@g0dkar). É minha forma de contribuir :)
